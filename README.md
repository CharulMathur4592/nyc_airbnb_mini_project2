**Project Overview**

This project analyzes the New York City Airbnb Open Data to explore hosting activity, pricing patterns, neighborhood dynamics, and factors influencing listing performance across the city. 
Using publicly available datasets, we investigate trends in Airbnb listings for New York City and generate insights to support data-driven decision-making for hosts, policymakers, and analysts.

**Dataset Description**

The primary dataset comes from Inside Airbnb, a project that provides publicly scraped Airbnb data for various cities around the world. The dataset includes detailed information for listings in New York City, such as:

1. Listing ID
2. Host information
3. Neighborhood and Neighbourhood groups
4. Room type
5. Price per night
6. Availability (days per year)
7. Number of reviews
8. Review scores
9. Geographic coordinates (latitude, longitude)

The dataset used in this project represents Airbnb listing activity in NYC for 2019.
